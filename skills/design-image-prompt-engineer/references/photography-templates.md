# Photography Templates

本文件只提供通用骨架和题材例外。摄影三轨、锚点预算、成像修饰器和主导光源以 [photography-framework.md](photography-framework.md) 为唯一来源，使用模板时不要复制或重新解释那套规则。

## 使用规则

1. 先按摄影框架选择一条轨道，再选择最接近题材的骨架；用户只说“像实拍 / 不要 AI 味”时不要自动改成抓拍。
2. 替换方括号并删除不适用项；模板是检查清单，不是必须逐项输出的格式。
3. 人物身份参考先读取 [person-reference.md](person-reference.md)，再补摄影部分。
4. 用户指定手机、胶片、一次性相机、早期 CCD 或扫描底片时，最多选一个成像修饰器并计入框架预算。
5. 需要词库时按需读取布光、构图、调色或风格库，不把所有术语堆进一条 Prompt。

## 通用骨架

```text
[主体、外观/材质、姿态/状态与交互]，位于 [场景、时间、天气与必要道具]，
[用途与画幅]，[主构图、景别、机位、焦段、拍摄距离与景深]，
[一个可解释的主导光源、环境光关系、阴影与曝光]，
[关键材质、反射、皮肤或空间细节]，[色彩与后期方向]，
[所选摄影轨道的收尾]，[可选且唯一的成像修饰器]
```

## 人物与活动

### 肖像 / 企业头像 / 时尚

```text
[人物、服装、表情与动作]，[背景或环境]，[头肩/半身/全身构图]，
[平视或明确机位]，[合理人像焦段与拍摄距离]，[面部主导光源]，
[肤色、服装材质与背景关系]，[轨道收尾]
```

- 企业头像、职业肖像、证件式肖像和正式合影可以直视镜头，不要自动改成生活抓拍。
- 普通生活照或安静生活方式人像默认稳定姿势；只有用户明确要求动作瞬间、未摆拍或纪实时才加入抓拍语言。
- 时尚任务写清服装结构、姿态、造型、系列用途和裁切安全区。

### 婚礼 / 活动 / 运动 / 街头

```text
[活动主体与正在发生的时刻]，[场地、时间和氛围]，[中景/全身/环境构图]，
[合理焦段与动作表现]，[一个现场主导光源]，允许真实环境光形成合理混合色温，
[情绪、色彩与轨道收尾]
```

- “婚礼抓拍”才进入抓拍轨道；正式合影可以使用稳定构图。混合光不等于多个互相矛盾的主光。
- 运动模糊、粉尘、汗水或 `mid-motion` 只在动作因果成立时使用，并纳入锚点预算。
- “街头”不等于人物必须避开镜头；视线服从叙事。

## 商品与商业内容

```text
[产品、食物、珠宝、美妆或静物及其材质状态]，置于 [台面/背景/使用场景]，
[主图/场景图/细节图与画幅]，[角度、标准/微距镜头、焦点范围]，
[主光、反射边界、接触阴影与背景分离]，[准确色彩与克制后期]，[轨道收尾]
```

- 产品真实拍摄保持台面、比例、反射和材质可信，不自行增加杂物、污渍、噪点、过曝或死黑。
- 美妆与皮肤特写写自然肤质或包装材质，不把“真实”解释为主动制造瑕疵。
- 珠宝、腕表和微距产品可使用必要的焦点堆叠；食物需写清可食用质感、温度和份量。
- 准确品牌、Logo 或包装文字按用户提供内容、位置和层级呈现；未要求时不例行排除正常文字。

## 汽车、空间与风景

### 汽车

```text
[车辆、漆面与状态] 位于 [道路/影棚/城市/自然环境]，
[静态英雄图/侧面追拍/三分之四视角]，[控制比例的标准至中长焦]，
[车身曲面、环境反射、主导光源与天气]，[轨道收尾]
```

汽车实拍强调车身比例、漆面与环境反射因果，不自动套用 candid 或主动制造脏污。

### 建筑 / 室内 / 房产 / 风光

```text
[空间或自然场景、尺度、动线/前中后景]，[时间、天气与陈设]，
[对称/一点透视/环境构图]，[不过度拉伸的合理广角或长焦压缩]，
[垂直线、空间比例、窗内外或天空地面亮度关系]，[材质、色彩与克制后期]
```

- 建筑和房产不使用抓拍语汇；允许合理包围曝光与克制合成，避免夸张 HDR、边缘光晕、高光全溢出或暗部死黑。
- 风光按天气、大气透视和尺度选焦段，不主动添加设备缺陷。

## 海报与叙事画面

广告、社媒和横幅需写画幅、留白与裁切安全区；准确主标题按用户文案呈现。电影式场景需写叙事动作、前中后景、景别、一个叙事主导光源及可信环境光，不把“电影感”当作空泛收尾。
