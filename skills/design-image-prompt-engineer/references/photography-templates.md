# Photography Templates

摄影任务使用“基础模板 + 一条摄影轨道 + 可选成像修饰器”组合，不为每种题材复制写实变体。先完成基础模板，再只选择一条轨道；用户明确指定设备或媒介时才叠加一个成像修饰器。

## 组合规则

1. 按题材选择一个基础模板，替换方括号内容。
2. 从“统一摄影轨道”选择且只选择一条收尾。用户只说“像实拍、不要 AI 味、真实拍摄”时默认选“干净真实”，不要自动改成生活抓拍。
3. 普通“生活照 / 生活方式人像”默认选干净真实；用户明确说“抓拍、随手拍、纪实、未摆拍、动作瞬间”时才选生活抓拍，并只在此轨使用 `subject unaware`、`mid-motion` 等抓拍措辞。
4. 用户明确指定手机、胶片、一次性相机、早期 CCD 或扫描底片时，叠加一个对应成像修饰器；不要同时堆叠多个预设。
5. 统计整条 Prompt 中的真实感锚点。基础模板或用户原 Prompt 已有的颗粒、轻微运动模糊、轻微偏色等都要计入总预算，不得在收尾叠加到 3 个或更多。一个完整成像预设计为一个锚点，而不是按预设内部短语逐个计数。
6. 商业精修默认不添加瑕疵锚点；干净真实使用 0–1 个；生活抓拍整条 Prompt 合计使用 1–2 个。用户指定的目标优先，但仍删除重复和互相冲突的锚点。
7. 干净真实与生活抓拍采用一个可解释的主导光源，可保留现实存在的环境光或弱辅助光。商业精修可使用受控多灯。不要把“一个主导光源”误写成绝对只有一盏灯。

## 统一摄影轨道

| 轨道 | 适用条件 | 统一收尾与约束 | 锚点预算 |
| --- | --- | --- | --- |
| 商业精修 | 广告、品牌大片、棚拍、电商主图、精确后期；或用户没有要求写实时的明确商业任务 | 受控单灯或多灯塑形，干净材质与边缘，精确曝光和色彩，克制而精细的商业后期；不要自行添加污渍、杂物、噪点或过曝 | 默认 0 |
| 干净真实 | “像实拍、真实拍摄、不要 AI 味、自然皮肤”，以及产品、建筑、房产、汽车、风光等非抓拍写实任务 | 合理镜头与透视，可信材质、曝光和阴影，一个可解释的主导光源，可有真实环境光或弱辅助光，克制后期；按需加入 0–1 个轻微真实感锚点 | 0–1 |
| 生活抓拍 | 仅限用户明确要求抓拍、随手拍、纪实、未摆拍、动作瞬间或决定性瞬间 | `candid documentary`，按场景选 `subject unaware` 或 `mid-motion`，一个可解释的现场主导光源并保留合理环境光，非摆布式瞬间，克制后期 | 整条 1–2 |

补充约束：企业头像、职业肖像、证件式肖像或正式合影可以直视镜头；“真实”不等于“不看镜头”。产品、建筑、房产、汽车和风光使用干净真实路径时，不添加 `subject unaware`、`mid-motion` 或生活杂物。

## 成像修饰器

每次最多选择一个预设；每个预设整体计为一个真实感锚点。预设已经包含成像链路，不要再拆出颗粒、偏色、锐化等重复锚点。

| 预设 | 可叠加的统一成像链路 |
| --- | --- |
| 手机 | 当代手机计算摄影直出，自动曝光与白平衡，合理的多帧融合，轻微 JPEG 与边缘锐化特征，不过度 HDR、不过度数字锐化 |
| 胶片 | 指定胶片感光与显影逻辑，自然颗粒、柔和高光过渡、可信色彩偏移，克制扫描或印放质感 |
| 一次性相机 | 小型塑料镜头、近距离硬闪或自然光、有限动态范围、消费级冲印色彩，偶发轻微失焦 |
| 早期 CCD | 早期数码便携相机直出，小型传感器色彩、直接机内处理、轻微高光截断与 JPEG 痕迹 |
| 扫描底片 | 底片扫描链路，适度颗粒与片基色彩、轻微扫描尘点或边缘痕迹，保留胶片高光层次 |

## 人物与活动

### Portrait / Corporate Headshot

```text
[人物身份与外观]，[表情、姿态与服装]，位于 [场景或背景]，
[半身/近景/环境肖像构图]，[机位、焦段与景深]，
[灯光结构与主导光源]，[肤色与背景色关系]，
[是否直视镜头：企业头像、职业肖像和证件式肖像可自然直视]，
[选择的摄影轨道收尾]，[可选成像修饰器]
```

用户提供人物参考照片并要求同一人物时，先按 [person-reference.md](person-reference.md) 声明照片职责、匹配取景并执行表情主权，再套用本模板。

```text
[姿态、动势与构图]，[焦段、机位与景深]，
[灯光结构]，[杂志/品牌美学]，[色彩与后期方向]，
[选择的摄影轨道收尾]，[可选成像修饰器]
```

### Wedding / Event

```text
[新人或活动主体]，处于 [仪式、合影或互动时刻]，
[中景/全身/环境构图]，[35mm/50mm/85mm 等合理焦段]，
[一个现场主导光源]，允许保留真实存在的室内灯、窗光或弱辅助光形成合理混合光，
[情绪、动作与色彩方向]，[选择的摄影轨道收尾]，[可选成像修饰器]
```

明确“婚礼抓拍”时才选择生活抓拍轨道；正式合影可使用干净真实并允许看镜头。混合光不等于多个互相矛盾的主光。

### Sports / Fitness

```text
[运动员或健身主体] 正在 [动作]，位于 [场地]，
[低角度/动态近景/环境构图]，[35mm/50mm 等焦段]，
[主导光线与体积关系]，[高速凝固或运动表现]，
[汗水、粉尘或空气细节，仅在场景真实存在时加入]，

```

若基础描述已经写入轻微运动模糊或 `mid-motion`，将其计入锚点预算。

### Street / Documentary

```text
[街头主体与城市场景]，[动作或决定性瞬间]，
```

## 空间与场景

### Landscape

```text
[地点或自然场景]，[时间、天气与大气条件]，
前景 [元素]，中景 [元素]，背景 [元素]，
[广角/长焦及透视关系]，[景深与对焦范围]，

[选择的摄影轨道收尾]，[可选成像修饰器]
```

### Architecture / Interior

```text
[建筑或室内空间]，[设计风格、尺度、动线与材质]，
[对称/一点透视/环境构图]，[合理广角与垂直线控制]，
[窗光或室内灯作为主导光源，并保留真实环境光]，
[真实材质、空间比例与克制后期]，[选择的摄影轨道收尾]，[可选成像修饰器]
```

建筑写实不套用 candid；允许为保留窗内外层次使用合理包围曝光与克制合成，不强制高光溢出或暗部死黑。

### Real Estate

```text
[空间类型、设计风格与陈设]，[尺度与空间动线]，
[一点透视/广角环境构图]，[合理焦段、真实空间比例与垂直线控制]，
[窗光、室内灯与环境光的可信平衡]，[明亮但自然的曝光与材质]，
[选择的摄影轨道收尾]，[可选成像修饰器]
```

真实房产照片允许合理包围曝光和克制合成以呈现人眼可接受的动态范围；不要强制有限动态范围、高光溢出、暗部死黑或夸张 HDR。
