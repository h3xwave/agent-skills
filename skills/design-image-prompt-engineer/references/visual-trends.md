# Visual Trends

这些趋势只作为可选灵感（截至 2026 年整理，请按需更新）。除非用户要求“更当下/更流行/赶潮流”，不要默认加入。按媒介取用：摄影类需求看「摄影向趋势」；插画、平面、矢量、3D、动态等非摄影需求看「插画 / 平面 / 3D / 动态向趋势」，不要把摄影趋势词套到非摄影画面上。

## 摄影向趋势

### Human Texture Over Perfection

- 方向：减少塑料感和过度磨皮，保留真实皮肤、材质瑕疵、现场光和瞬间感。
- Prompt 语言：natural skin texture, imperfect realism, candid timing, tactile surface detail, subtle grain。
- 适用：人像、生活方式、旅行、品牌故事、社媒 campaign。

### Sensory And Emotional Imagery

- 方向：强调触感、温度、声音联想、气味和情绪连接，而不只是“漂亮画面”。
- Prompt 语言：tactile fabric texture, warm humid air, soft ambient noise implied, intimate mood。
- 适用：美妆、香氛、食品、酒店、家居、生活方式。

### Playful Surreal Realism

- 方向：真实摄影逻辑中加入轻微不可能元素，如尺度变化、悬浮、错位、重复。
- Prompt 语言：photoreal surreal editorial, impossible but physically lit, playful scale shift。
- 适用：广告、时尚、社媒视觉、创意产品图。

### Local And Cultural Specificity

- 方向：避免空泛全球化视觉，加入真实地域材料、生活方式、空间纹理和日常细节。
- Prompt 语言：locally specific materials, lived-in environment, authentic cultural context。
- 适用：旅行、餐饮、城市品牌、地方文创、酒店和零售。

### Analog / Retro Digital

- 方向：胶片、直闪、扫描质感、Y2K、早期数码相机和低保真视觉回潮。
- Prompt 语言：direct flash, 35mm film grain, compact digital camera look, slight motion blur。
- 适用：时尚、音乐、夜生活、年轻品牌、纪实 campaign。

### Quiet Luxury And Material Honesty

- 方向：减少浮夸装饰，强调材质、比例、留白、低饱和色和精确光线。
- Prompt 语言：quiet luxury, controlled highlights, restrained palette, premium material detail。
- 适用：珠宝、腕表、香氛、家居、汽车、建筑室内。

## 插画 / 平面 / 3D / 动态向趋势

### Maximalist And AI-Native Illustration

- 方向：高饱和、超叠层、密集叙事的极繁插画，拥抱 AI 原生的超现实拼贴与不可能空间。
- Prompt 语言：maximalist illustration, dense layered composition, hyper-saturated palette, AI-native surrealism, impossible dreamscape。
- 适用：唱片/演出视觉、潮流品牌、杂志插画、社媒大图、活动主视觉。

### Soft 3D / Claymorphism / Bento

- 方向：圆润柔和的 3D、黏土与软体材质、bento 便当式分格排版，亲和、可爱、产品化；是渲染而非实拍。
- Prompt 语言：soft 3D render, claymorphism, rounded inflatable forms, matte clay material, bento grid layout, soft global illumination。
- 适用：科技品牌、App/UI 视觉、图标系统、说明性插画、电商 banner。

### Anti-Design / Brutalist Graphic

- 方向：刻意“未精修”的粗野排版、超大字阶、硬边网格、做旧复印质感，反精致美学。
- Prompt 语言：brutalist graphic design, anti-design layout, oversized abstract type blocks, raw grid, xerox texture, high-contrast monochrome accents。
- 适用：文化海报、独立刊物、音乐厂牌、潮流平面、展览视觉。

### Risograph And Print Revival

- 方向：risograph / 丝网套色、半调网点、套色错位、有限专色与纸面油墨颗粒的手作印刷感回潮。
- Prompt 语言：risograph print, spot-color overprint, halftone dots, misregistration, limited ink palette, grainy paper texture。
- 适用：独立出版、海报、文创周边、活动物料、年轻文化品牌。

### Glitch / Databending

- 方向：扫描线、RGB 色偏、像素撕裂、数据损坏块与 CRT 失真构成的数字破碎美学。
- Prompt 语言：glitch art, databending, scanlines, RGB channel shift, pixel sorting, datamosh, CRT distortion。
- 适用：电子音乐、科技/赛博主题、流媒体封面、实验视觉、游戏。

### Y2K Chrome And Gradient-Mesh Vector

- 方向：液态铬金属、渐变网格、星芒高光、半透明玻璃与 Y2K 复古未来感的矢量视觉。
- Prompt 语言：Y2K aesthetic, liquid chrome, gradient mesh vector, glossy bubble shapes, lens-flare star highlights, iridescent gradients。
- 适用：潮流/美妆品牌、音乐视觉、社媒、时尚、科技活动。
