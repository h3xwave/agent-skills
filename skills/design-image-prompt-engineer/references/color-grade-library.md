# Color Grade Library

把调色作为摄影轨道之后的色彩层。说清色彩科学、对比、颗粒和适用题材；不要只写“电影感 / 胶片感”，也不要把非摄影需求强行套入胶片与镜头词。

## 轨道适配

- 商业精修：优先准确肤色或产品色、受控对比、干净高光与精确局部调整；胶片外观只在品牌方向需要时使用。
- 干净真实：优先自然白平衡、可信局部对比、适度机内锐化与克制降噪；不要为了“照片感”自动加入颗粒、漏光或褪色。
- 生活抓拍：让色彩响应符合设备和现场光；所选手机、胶片、CCD 等完整成像预设整体算一个真实感锚点。
- 避免过度 HDR、过度数字锐化、边缘光晕与统一套色；不要把正常的手机计算摄影、JPEG 直出锐化或合理曝光合成当作错误。

## 彩色胶片仿真

选择一款胶片外观时，把其颗粒、色彩响应与高光特征作为一个成像预设计入锚点预算；不要再逐项重复叠加。

- Kodak Portra：柔和肤色、低至中等对比、平滑高光与自然暖调；适合人像、婚礼、生活方式和时尚。
- Fujifilm Pro 400H：轻微冷绿、柔和高光与清透肤色；适合婚礼、清新人像和户外生活方式。
- Kodak Gold 200：暖黄怀旧、轻微泛红与日常颗粒；适合旅行、街拍和复古生活方式。
- Cinestill 800T：钨丝灯平衡、高光红晕 halation 与夜景冷暖关系；适合城市、霓虹和音乐视觉。
- Fuji Velvia：高饱和绿蓝与强对比；适合风光和自然，谨慎用于肤色与产品准确色。
- Kodak Ektar 100：细腻、较高饱和与清晰色彩；适合风光、产品和旅行。
- Cross-process / Lomo：明显偏色、强对比和不可预期感；用于实验方向，不用于一般“像实拍”请求。

## 数字调色 / LUT

- Teal-Orange：暖肤色与青蓝背景的互补关系；适合广告、电影感人像和汽车，避免过度统一套色。
- Bleach bypass：低饱和、高对比与银盐厚重感；适合硬朗叙事和纪实电影感。
- Moody desaturated：低饱和、压暗与冷调情绪；适合悬疑、奢侈品和深色人像。
- Warm filmic：暖调、柔和对比与胶片化曲线；适合生活方式、品牌故事和温馨场景。
- Clean commercial：中性白平衡、准确肤色或产品色、受控对比；适合电商、企业头像和产品主图。
- Natural JPEG：自然自动白平衡、适度机内锐化、有限但不刻意压缩的动态范围；适合干净真实和手机直出。
- Pastel / faded：低对比褪色、柔粉调与轻盈感；适合美妆、青春和清新品牌。

## 黑白

- High-contrast B&W：强黑白与深阴影；适合时尚、肖像和街头，保留关键主体层次。
- Documentary B&W：丰富中灰、自然颗粒与克制反差；适合纪实、新闻和人文。
- Fine-art B&W：长调控制与细腻影调；适合艺术风光、建筑和静物。
- Silver gelatin B&W：银盐颗粒与暗房质感；适合复古肖像、纪实和艺术方向。

## 处理痕迹 / 工艺

- Halation：高亮边缘出现红橙扩散与柔焦光晕；只在相应胶片、强光与曝光因果成立时使用。
- Light leaks：边缘暖色漏光或随机色斑；作为独立锚点使用，不要自动加到所有胶片图。
- Grain structure：按胶片尺寸、感光度和曝光控制颗粒密度；不要同时要求胶片颗粒与传感器彩噪。
- Faded matte：抬高黑位、降低对比与轻微褪色；适合复古和生活方式，不用于需要准确黑位的产品或建筑。
- Bracketed architectural blend：允许房产、室内和建筑使用合理包围曝光与克制合成；保持窗内外亮度关系、自然局部反差和干净边缘，避免过度 HDR。
- Restrained sharpening：让锐度符合镜头、设备和观看尺寸；保留自然纹理，避免过度数字锐化造成的白边与塑料细节。
