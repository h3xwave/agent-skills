# Photorealism（摄影轨道与真实感控制）

把本文件作为摄影写实语义的权威来源。先选摄影轨道，再叠加成像修饰器；不要把“像实拍 / 不要 AI 味”自动解释为抓拍、低质或脏乱。

## 三轨路由

| 摄影轨道 | 进入条件 | 默认真实感锚点预算 | 允许的处理 |
|---|---|---:|---|
| 商业精修 | 用户要广告、品牌 campaign、电商主图、杂志美妆、奢侈品 hero shot 等 | 0 | 受控多灯、精确反射、干净材质、焦点堆叠、精细但自然的后期；不要为“真实”主动加瑕疵 |
| 干净真实 | 用户说“像实拍 / 照片感 / 不要 AI 味 / 自然皮肤”，或只要求可信照片 | 0–1 | 合理镜头、可信材质、自然曝光与阴影、克制后期；可加一个轻微锚点，也可完全不加 |
| 生活抓拍 | 用户明确说抓拍、随手拍、纪实、未摆拍、偷拍视角、派对快照或动作瞬间等 | 1–2 | 现场感、瞬时动作与轻微不完美；只按用户意图使用 `subject unaware`、`mid-motion` |

默认规则：普通生活照或生活方式人像，以及企业头像、证件式肖像、产品、建筑、房产、汽车、风光等，即使要求“真实”，也先走“干净真实”；不要套用 candid。企业头像、证件式肖像和正式人物照可以自然直视镜头。

## 真实感锚点总预算

真实感锚点是为了打破合成感而主动写入的成像签名、现场微瑕、环境生活痕迹或瞬时语言。镜头焦段、主体材质、正常曝光、可信透视等基础摄影信息不算锚点。

按整条 Prompt 计数，而不是按章节计数：

- 把用户 brief、所用模板已有词、成像修饰器和后续补充全部纳入同一预算；模板已经有锚点时先扣除，不得叠加到 3 个或更多。
- 每个独立短语算一个锚点，例如 `subject unaware`、`mid-motion`、轻微手震、背景生活痕迹各算一个。
- 一个完整成像预设整体算一个锚点，即使预设内部包含多项彼此配套的成像特征。
- 商业精修默认 0 个；干净真实使用 0–1 个；生活抓拍使用 1–2 个。用户明确要求的特征优先占用预算。
- 用户明确指定胶片、手机等成像预设时，可以覆盖商业精修的默认 0；把该预设计为 1 个锚点，并停止主动添加其他真实感瑕疵。
- 达到预算后停止添加；不要为了“更真实”同时加入噪点、过曝、杂乱、倾斜、失焦和划痕。

```text
手机婚礼抓拍：手机成像预设（1）+ mid-motion（1）= 2；不再加噪点或过曝。
护肤品像实拍：可信瓶身反射与自然阴影属于基础信息，可保持 0 个锚点。
```

## 成像修饰器

成像修饰器可叠加在任一摄影轨道上，但不改变轨道本身。通常只选一个预设，并把整组特征算作一个锚点。

- 手机直出：轻广角、计算摄影曝光、适度机内锐化、自然 JPEG 质感；不要同时要求低动态范围或全面高光溢出。
- 35mm 胶片：自然颗粒、柔和高光过渡、轻微色偏；按指定胶片与场景调整，不要自动加划痕。
- 一次性相机：固定焦段、直接闪光、较粗颗粒、有限控制；适合旅行、聚会和青春记录。
- 早期 CCD：消费级 JPEG 色彩、有限分辨率、可能的直闪或高光溢出；只在复古或年代意图明确时使用。
- 扫描底片：胶片色罩、扫描颗粒，可有轻微灰尘或边缘痕迹；不要把所有老化损伤一起堆入。
- 一次成像：轻微软化、化学显影色偏、相纸边框；适合私人记忆感。

## 可选锚点词库

只在轨道预算内选用，且必须符合场景因果。

### 现场微瑕

- 合理混合色温：一个主导光源下，现场钨丝灯、窗光或霓虹形成轻微色温污染。
- 轻微白平衡偏差：适合消费级 JPEG 或复杂现场光，不用于要求准确色彩的产品图。
- 镜头眩光 / 鬼影：仅用于真实逆光方向。
- 轻微手震或运动模糊：仅用于慢门、暗光或运动瞬间。
- 局部高光溢出或阴影沉底：仅在曝光取舍合理时使用；不要强加给房产、建筑或商业产品。
- 高 ISO 噪点或胶片颗粒：按设备、光照与感光度选择，不要两者并用。

### 环境生活痕迹

- 一处轻微使用痕迹、自然摆放物、远处路人或天气痕迹可增强现场感。
- 产品、房产、汽车和建筑的干净真实图默认不添加杂物、指纹、灰尘或破损。

### 瞬时语言

- `subject unaware`：只用于明确抓拍且主体确实未察觉镜头的情境。
- `mid-motion`：只用于动作进行中的抓拍；静态头像和正式合影不使用。
- 不完美裁切、轻微偏轴、前景偶然遮挡：每项单独计数，只在随手拍意图明确时使用。

## 可解释的主导光源

干净真实与生活抓拍都要有一个可解释的主导光源，例如窗光、阴天漫射、顶灯、路灯、直接闪光或摄影师设置的主灯。允许真实存在的环境光、实景灯具和弱辅助光，只要它们不争夺主导关系。

- 婚礼、餐厅、街景等混合光环境：说明谁主导曝光，再保留现场色温差；不要假装只有一个物理光源。
- 干净真实：可用弱反光板或低功率补光控制反差，但保持阴影方向可信。
- 生活抓拍：优先现场光或直接闪光，不要堆满主光、补光、轮廓光、发丝光。
- 商业精修：允许受控多灯与复合反射；每盏灯仍需有明确作用。

## AI 味诊断

- 塑料皮肤 / 过度磨皮 → 写自然皮肤纹理与克制修饰；不要自动加入痘印、油光和杂毛全套。
- 过度 HDR（局部反差处处相同）→ 使用自然明暗层级与克制色调映射；建筑、房产可合理包围曝光并克制合成。
- 过度数字锐化 / 边缘光晕 → 使用自然细节频率与适度机内锐化；手机 JPEG 的正常计算摄影不需要被排除。
- 图库式人物姿态 → 先判断用途；企业头像可看镜头，只有明确抓拍才用 `subject unaware` 或 `mid-motion`。
- 不可信的材质与反射 → 写清表面粗糙度、反射形状与接触阴影，而不是加灰尘或磨损。
- 无来源的均匀亮 → 确定主导光源、方向、衰减和阴影，再决定是否有环境光或弱辅助光。
- 建筑空间的假 HDR → 保持窗外与室内亮度关系、自然局部对比和垂直线；允许合理包围曝光，不强制高光溢出或暗部死黑。
