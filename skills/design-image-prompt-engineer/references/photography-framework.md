# Photography Framework

构建摄影 Prompt 时，先定摄影轨道，再组合主体、环境、构图、光线、镜头和后期。不要机械罗列所有项，也不要把所有摄影需求写成抓拍。

## 组装顺序

1. 根据用途选择“商业精修 / 干净真实 / 生活抓拍”；“像实拍 / 不要 AI 味 / 普通生活照 / 生活方式人像”默认进入干净真实，只有明确抓拍、随手拍、纪实、未摆拍或动作瞬间才进入生活抓拍。
2. 写清主体、空间关系和画面用途，再选择合理镜头、主导光源与材质表现。
3. 需要手机、胶片、一次性相机、早期 CCD 或扫描底片时，叠加一个成像修饰器。
4. 按 `photorealism.md` 统计整条 Prompt 已有的真实感锚点；模板、用户要求和成像预设都计入总预算。
5. 检查人物视线、光源因果、曝光与后期是否符合所选轨道。

## 摄影轨道

| 轨道 | 典型用途 | 光线与后期 | 构图与主体 |
|---|---|---|---|
| 商业精修 | 广告、品牌 campaign、电商、美妆、奢侈品 | 可用受控多灯、精确反射、焦点堆叠和干净精修 | 构图可高度设计，不主动加瑕疵 |
| 干净真实 | 企业头像、产品实拍、建筑、房产、汽车、风光，以及一般“像实拍”请求 | 一个可解释的主导光源，可有环境光或弱辅助光；自然曝光与克制后期 | 人物可看镜头；物体与空间保持整洁可信 |
| 生活抓拍 | 明确的抓拍、随手拍、纪实、未摆拍、派对快照或动作瞬间 | 现场光或直接闪光优先，保留合理瞬时感 | 只按意图使用 `subject unaware`、`mid-motion` 或偶然构图 |

## Subject

- 写主体类型：人物、产品、建筑、室内、自然景观、食物、交通工具或抽象物。
- 写可见细节：年龄与外观、姿态与动作、材质与表面纹理、比例、表情、交互关系。
- 明确焦点优先级：主焦点、辅助元素、需要弱化或排除的元素。
- 匹配视线与用途：企业头像、证件式肖像、正式合影和品牌肖像可以自然直视镜头；不要为了真实感强迫人物避开镜头。
- 保持非人物题材干净：产品、建筑、房产、汽车、风光不使用人物抓拍语汇，除非用户另有叙事要求。

## Environment

- 写场景、时间、天气、季节、背景层次、空间尺度和必要道具。
- 根据用途选择清晰叙事背景、极简背景、真实室内、浅景深环境或前中后景层次。
- 干净真实不等于杂乱：只保留有因果的环境细节，不自动加入垃圾、灰尘、指纹或无关路人。
- 建筑与房产保持空间真实、窗内外亮度关系和陈设秩序；允许克制的曝光合成。

## Composition

- 从中心、三分、对角线、对称、框中框、负空间和前景层次中选择最适合用途的一种主结构。
- 写机位与景别：平视、低角度、高角度、鸟瞰、过肩、特写、全景或环境广角。
- 按交付选择画幅：1:1 主图、4:5 社媒、3:2 摄影、16:9 横幅、9:16 竖屏。
- 商业精修和干净真实可以精确构图；只有生活抓拍且用户有此意图时才使用偏轴、不完美裁切或前景偶然遮挡。

## Lighting

- 指定一个可解释的主导光源、方向、光质、衰减和主要阴影。
- 允许真实存在的环境光、实景灯具或弱辅助光；混合光场景要明确主导曝光与色温关系。
- 商业精修可使用主光、补光、轮廓光和反射控制，但每盏灯都要服务材质或轮廓。
- 干净真实与生活抓拍不要堆满影棚灯位；生活抓拍优先现场光或直接闪光。

## Lens And Camera

- 人像：50mm–85mm 常用于自然或压缩透视；环境人像可用 35mm。焦段与拍摄距离优先于盲目追求极浅景深。
- 产品：50mm–100mm 标准或微距镜头；按用途选择单点清晰或克制焦点堆叠，保持边缘与反射可信。
- 建筑 / 房产：合理广角、透视校正、垂直线与空间尺度可信；避免不必要的超广角拉伸。
- 汽车：标准至中长焦控制车身比例，运动场景再加入合理快门与拖影。
- 风光：广角或长焦压缩都可，按前中后景、天气与尺度选择；不要默认加入抓拍瑕疵。
- 运动 / 纪实：35mm 或 50mm 可提供自然透视；只有动作与快门逻辑需要时才加入运动模糊。

## Style And Post

- 商业精修：准确肤色或产品色、精确高光、干净材质、自然但精细的修饰。
- 干净真实：自然皮肤与材质、可信局部对比、适度锐化、不过度 HDR、不过度磨皮。
- 生活抓拍：保留符合设备和现场的轻微瞬时感，不把低质量当作真实本身。
- 建筑 / 房产：允许合理包围曝光与克制合成，保持自然窗景、局部对比和色彩；不要强制高光溢出或暗部死黑。
- 需要胶片、电影调色或设备直出时，使用 `color-grade-library.md`；选择风格时使用 `style-library.md`。
