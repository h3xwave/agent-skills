# Non-Photographic Templates

先选择媒介，再替换方括号内容。非摄影模板使用版式、观察角度、空间层级、上色、描边、印刷或渲染语言；不要自动加入焦段、光圈、摄影景深、胶片颗粒、相机型号或摄影轨道。只有用户明确要求伪摄影 CGI 或混合媒介时才引入相应摄影元素，并说明它们的媒介作用。

### Flat Vector Illustration

```text

```

### Graphic Poster

```text
[主题] 平面设计海报，[网格/对角/满铺] 版式，
[限定撞色色板]，大色块、抽象字形与负空间构成，
[半调/丝网印刷/干净矢量] 质感，[信息密度与视线顺序]，
文字内容按用户目标处理；未指定文案时可使用符合主题的简短辅助文字或抽象字形，不默认排除可读文字；用户提供文案时写明准确内容、位置与层级
```

### Isometric Infographic

```text
[系统、流程或空间主题]，等距轴测信息图，统一轴线与无消失点投影，
[模块化场景与分层结构]，[限定色板与统一图标]，

标签可按信息表达需要使用清晰短文字；用户提供标签时写明准确文字和对应模块
```

### Icon / UI System

```text
[图标主题集合]，矢量图标系统，统一栅格、圆角和描边粗细，
[线性/面性/双色] 语言，[2–3 色品牌色板]，
几何简化、光学尺寸与视觉重量一致，系列化且可扩展，
干净 SVG 式边缘；品牌标志按用户目标或场景语境处理并允许正常出现，要求准确还原时以用户提供的素材为准
```


### Glitch Graphic

```text
[主体或信息主题]，故障图形设计，[像素位移/色频分离/数据弯折]，
[扫描线、压缩块或界面碎片]，[限定数字色板]，
[版式层级与故障密度]，有意的数字错误形成清晰视觉节奏，

```

### Double-Exposure Mixed Media

```text
[基底主体或场景] 与 [线稿、色块或纹理层] 双重曝光式叠合，
半透明交融、遮罩边缘与虚实层次，[色调渐变与情绪]，
[低信息密度留白/层次化满铺]，统一纸面或数字混合媒介质感，
叠加层服务主体轮廓，不复制具体参考图情节
```
