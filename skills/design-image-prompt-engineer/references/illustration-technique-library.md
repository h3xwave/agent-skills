# Illustration Technique Library

构建非摄影画面时，描述要落到具体手法（怎么上色、怎么描边、什么印刷或数字痕迹、什么渲染），不要只写"插画风"。非摄影画面避免误用镜头型号、景深、胶片颗粒等摄影词，除非用户明确要伪摄影 CGI 或混合媒介。每条给"手法 + 适合"，与 `non-photo-style-library.md` 的流派定调配合使用。

## 上色 / 渲染法

- Cel shading：平涂分层加硬边单层投影，描边收口，色块明确；适合动漫、角色插画、表情包、游戏立绘。
- Lineless：无描边，靠色块与明暗边界造形，柔和现代；适合品牌插画、编辑配图、儿童、UI 配图。
- Painterly：可见笔触、混色过渡、手绘厚度，数字油画感；适合概念美术、封面、氛围插画、影视海报。
- Gouache 厚涂：不透明水粉质感、哑光平涂叠笔触、轻微纸纹；适合绘本、复古印刷、文创、编辑插画。
- Flat 2–3 色平涂：限定色、纯色块、零渐变、强识别；适合 logo 辅助图形、海报、图标、社媒签名视觉。
- Grain / 颗粒填充：在平涂上叠噪点与纸面颗粒，去数字塑料感；适合复古、riso、有温度的品牌、编辑。
- Halftone 网点上色：用网点疏密表现明暗与渐变，印刷复古；适合波普、漫画、海报、做旧视觉。
- Watercolor 水彩晕染：透明叠色、湿边渗化、留白透气；适合绘本、自然主题、轻柔品牌、节日。

## 线条 / 描边

- 均匀描边 uniform outline：等宽轮廓、干净图标感；适合扁平插画、UI、贴纸、图标系统。
- 粗细变化 tapered line：起收笔提按、书法感动势；适合编辑插画、国潮、手绘品牌。
- 断线 / 手抖 sketchy：断续抖动线、草图随性、人情味；适合绘本、涂鸦、年轻品牌、zine。
- 无描边 lineless edge：靠色块分界不用线；适合柔和现代插画、3D 平面化、品牌。
- 几何精确线 geometric：直线圆弧、网格对齐、构成主义；适合信息图、技术图解、Swiss 平面、logo。

## 印刷痕迹

- 套色错位 misregistration：分色版偏移露边、手工套印感；适合 riso、复古海报、独立出版、文创。
- 丝网叠印 screenprint overprint：专色叠加生成第三色、油墨厚度；适合潮牌、海报、乐队周边。
- CMYK 减色 / 分色：四色网点叠印、印刷原生质感；适合波普、杂志、复古商业视觉。
- 半调网点 halftone：点阵表现连续调、放大可见网点；适合漫画、波普、做旧、报纸感。
- 做旧纸面 grunge paper：泛黄、折痕、污渍、磨损边；适合复古、zine、街头、唱片视觉。

## 数字痕迹 / glitch

- 扫描线 scanlines：水平行线、CRT 显示器感；适合赛博、复古科技、游戏、实验视觉。
- RGB 色偏 channel shift：红绿蓝通道错位、立体重影；适合 glitch、音乐封面、科技、躁动视觉。
- 像素撕裂 pixel sorting：像素沿行列拉伸排序、数据流感；适合电子音乐、实验、科技艺术。
- Datamosh：帧间压缩损坏、融化拖影；适合 MV、流媒体封面、实验、前卫视觉。
- CRT 失真 distortion：弧面、暗角、摩尔纹、扫描卷动；适合复古游戏、vaporwave、怀旧科技。
- JPEG 压缩块 artifact：块状色斑、边缘振铃、低保真做旧；适合亚文化、meme、实验平面。

## 3D 渲染法

- Clay render：单色黏土材质、看体块不看材质、柔光；适合方案预览、claymorphism、说明性 3D。
- Toon shader：3D 几何套卡通描边与色阶，三维的二维感；适合游戏、动画、潮玩、混合风。
- SSS 软体 subsurface：次表面散射、半透光软质（蜡/皮肤/果冻）；适合食品、美妆、软体角色、产品。
- PBR 金属 / 玻璃：物理材质反射折射、真实高光；适合产品可视化、汽车、珠宝、科技。
- AO / GI 全局光：环境光遮蔽加全局光照、柔和真实的接触阴影；适合写实渲染、建筑、产品、室内。
- Low-poly 平面着色：低面数、硬边切面、flat shading；适合游戏、信息图、轻量 3D、科技。
