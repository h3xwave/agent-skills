# Non-Photo Style Library

从零构建非摄影画面时，按用户目标选少量流派定调，不要把多个不相容流派堆进同一条 prompt。可借艺术运动或时代风格定调，但不要把参考图中特有的受保护 IP 或完整品牌视觉当作风格模板照搬；正常场景中的品牌、Logo 和文字可自然出现。每条给"签名 + 适合"，命中后再去 `illustration-technique-library.md` 取具体技法词。

## 平面 / 版式流派

- Swiss / International Typographic：严格网格系统、无衬线、左对齐、大量留白、客观中性，图像与色块按网格排布；适合企业、文化机构、年报、极简海报。
- Bauhaus：基本几何（圆/三角/方）、原色加黑、构成主义排版、功能主义；适合艺术展、设计品牌、教育、复古现代海报。
- Constructivism 构成主义：红黑对角动势、几何切割、照片蒙太奇、粗体斜置字块；适合宣言式海报、活动主视觉、音乐厂牌。
- Art Deco：对称、阶梯几何、金黑配色、扇形与流线、奢华装饰线条；适合奢侈品、酒店、影视片头、复古请柬。
- Memphis：80 年代撞色、波点与折线、随机几何、玩味反精致；适合年轻品牌、潮流、儿童、活动视觉。
- Brutalist / Anti-design：粗野无修饰、超大字阶、硬边网格、系统字做旧、刻意"丑"；适合独立刊物、潮牌、展览、亚文化。
- De Stijl / Mondrian 风格派：水平垂直黑线分割、原色块填充、纯粹抽象；适合极简品牌、家居、艺术周边。

## 插画流派

- 扁平极简 flat：纯色块、几何简化、无渐变、图标式；适合 UI、说明图、品牌插画、社媒。
- 编辑插画 editorial：概念隐喻、风格化人物场景、杂志气质、有观点；适合报刊、专栏、观点长文、活动海报。
- 新中式 / 国潮：水墨笔意叠现代构成、矿物色、传统纹样再设计；适合茶饮、节庆、文创、品牌联名。
- 浮世绘平面分色：平涂分色、流动线条、波浪与云纹、有限木刻套色；适合和风、包装、海报、纹样。
- 绘本水彩 storybook：柔和水彩晕染、手绘温度、圆润角色、叙事场景；适合童书、教育、亲子品牌、节日。
- 波普 Pop Art：网点、撞色、漫画分格、商业符号挪用；适合潮流、音乐、快消、活动。
- 复古印刷插画 mid-century：50–60 年代纹理平涂、有限色、几何化自然形；适合咖啡、文创、品牌叙事、海报。
- 线描 line art：单色连续线、极简留白、优雅克制；适合美妆、轻奢、编辑配图、logo 辅助图形。

## 3D / CGI 流派

- Soft 3D / Claymorphism：圆润软体、磨砂黏土材质、柔和全局光、亲和可爱；适合科技品牌、App、图标、电商 banner。
- Low-poly：低多边形切面、平面着色、几何块面；适合游戏、科技、信息图、轻量插画。
- 盲盒 / toy render：可动人偶质感、塑料或搪胶材质、棚拍柔光、产品化；适合潮玩、IP 衍生、电商、收藏品。
- 等距 isometric：轴测无透视、模块化场景、清晰分层；适合信息图、流程图、城市/产品场景、SaaS 视觉。
- 写实 PBR：物理材质（金属/玻璃/木/布）、真实光照、高保真渲染；适合产品可视化、汽车、建筑、电商主图。
- 玻璃拟态 glassmorphism：半透明磨砂玻璃、背景模糊、彩色折射高光；适合 UI、科技海报、品牌 KV。
- 充气 inflatable / blobby：膨胀软塑形、橡胶光泽、圆角无棱；适合潮流、活动、3D 立体字、社媒大图。

## 复古 / 亚文化

- Y2K 金属 chrome：液态铬、星芒高光、渐变网格、千禧未来感；适合美妆、音乐、潮流、社媒。
- Vaporwave：粉紫青霓虹、古典雕塑配网格地平线、故障与点缀字符、怀旧反讽；适合音乐、海报、实验视觉。
- 像素 8-bit / pixel art：方块像素、有限调色板、抖动 dithering、复古游戏感；适合游戏、表情包、潮玩、科技怀旧。
- Cyberpunk graphic：霓虹高反差、HUD 与故障元素、雨夜城市色、高信息密度；适合科技、游戏、音乐、活动。
- Risograph：套色错位、专色油墨、半调网点、纸面颗粒手作感；适合独立出版、海报、文创、年轻文化品牌。
- Grunge / zine：撕贴拼接、复印噪点、做旧污渍、手写涂鸦、躁动反叛；适合音乐、滑板、街头、地下文化。
- Synthwave / Retrowave：80 年代日落渐变、霓虹网格、复古字体、合成器美学；适合音乐、游戏、健身、夜间活动。
