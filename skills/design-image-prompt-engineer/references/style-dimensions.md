# Style Dimensions

参考图风格分析时覆盖这些维度，并自然融入最终 Prompt。不要逐项输出分析过程。

1. 画面风格：整体艺术取向，如写实、插画、像素、3D、拼贴等。
2. 画面成分组成：前景、中景、背景的元素组织。
3. 构图方式：黄金分割、对称、对角线、中心、三分、框中框等。
4. 分镜类型：特写、中景、远景、俯拍、仰拍、鱼眼、广角、长焦等。
5. 光影特质：硬光或柔光、方向、对比度、伦勃朗光、轮廓光、体积光等。
6. 色调与色彩科学：冷暖、饱和度、互补或类似色、电影调色 LUT、印刷油墨感等。
7. 媒介与材质纹理：油画笔触、水彩、丝网印刷、CGI、胶片颗粒、布面纹理等。
8. 情绪与氛围：孤寂、躁动、温馨、史诗、赛博、怀旧等。
9. 渲染或拍摄参数：焦段、景深、快门感、Octane、Arnold、Unreal、采样数等。
10. 时代感与文化语境：70 年代复古、Y2K、昭和、Art Deco、新中式等。
11. 空间逻辑与透视关系：一点、两点、三点透视、平面化、轴测、错位等。
12. 信息密度与留白：极简、密集堆叠、负空间比例等。
13. 动态状态：凝固瞬间、动势模糊、慢门拖影、定格张力等。
14. 后期处理与数字痕迹：颗粒、漏光、色差、扫描线、JPEG 压缩、半色调网点等。
15. 符号化特征：该风格独有的、一眼可识别的视觉签名元素。

## Transfer Rules

- 抽取构图、光线、色彩、材质、媒介、空间、氛围和后期处理，不抽取原图身份。
- 参考图为非摄影媒介（插画/平面/矢量/拼贴/glitch/3D/动漫）时，内部分析一并参考 `illustration-technique-library.md` 命名具体技法（上色/描边/印刷/数字痕迹/渲染）与 `non-photo-style-library.md` 命名流派和时代语境，让媒介材质（第 7 维）与符号化签名（第 15 维）更精确；摄影参考图则参考 `lighting-techniques.md` 与 `color-grade-library.md`。最终仍只输出融合后的连贯 Prompt，不罗列维度或技法名。
- 删除或泛化具体人物、IP 角色、品牌、地名、可读文字、台词和原图情节。
- 用户要求保留参考照片中的人物身份（同一个人出现在新画面）时，属于人物参考一致性任务，改按 [person-reference.md](person-reference.md) 处理，不做身份剥离。
- 用户要求“换主体”时，把新主体放入完整 prompt（媒介随参考图，可能是摄影、插画、平面、3D 等）；用户要求“通用模板”时使用占位符。
- 如果新主体含具体 IP、名人、品牌角色或受保护视觉元素，将其泛化为原创主体描述；不保留角色名、编号、logo、专属服装、阵营符号、台词或可读文字。
- 不要把图像识别、OCR、人物鉴定或品牌确认伪装成风格分析。
