# Reference Image Style Output Contract

参考图风格模板是严格输出模式。用户要求“分析风格”“反推提示词”“提取图片风格”“生成通用 Prompt”“做风格模板”，或要求保留风格换主体但没有提供具体新主体时，先按 [style-dimensions.md](style-dimensions.md) 在内部分析，再执行以下契约。用户提供具体新主体并明确要立即可用的完整 Prompt 时，不使用本模板契约；改按风格维度生成完整 Prompt，并继续剥离原图身份。用户要求保留照片中人物身份（同一个人出现在新画面）时，属于人物参考一致性任务，改按 [person-reference.md](person-reference.md) 处理，不适用本契约的身份剥离规则。

## Required Output

1. 直接输出一段完整、高水准的中文提示词；不要前言、标题、分析、列表、解释或寒暄。
2. 在提示词开头或核心主体位置原样包含且只包含一次 `[在此处替换为您想要生成的主体内容]`。
3. 输出一段或最多两三段连贯散文式描述，不使用 Markdown 标题、代码块、项目符号或编号。
4. 不附负向提示词、平台参数或平台使用说明，即使用户同时点名平台；只有用户明确要求另附参数时才在模板之外处理。

## Content Coverage

将参考图的可迁移风格压缩进自然语言 Prompt，覆盖与画面有关的主体层次、前中后景、构图、景别与视角、光影、色彩科学、媒介与材质、情绪、空间与透视、信息密度、动态、时代语境、后期痕迹和符号化视觉签名。摄影参数、渲染器或技法只在与参考媒介匹配时加入，不逐项罗列分析维度。

## Identity Removal

- 删除或泛化原图特有的人名、IP 角色、品牌、商标、地标、可读文字、台词和具体故事情节，只保留可迁移美学。
- 新主体若包含名人、具体 IP、品牌角色或受保护视觉元素，也泛化为原创主体描述；不保留角色名、编号、logo、专属服装、阵营符号、台词或可读文字。
- 模板始终使用上述占位符，不同时写入具体新主体；具体新主体的完整 Prompt 属于本契约之外的参考图迁移模式。

## Final Check

- 占位符数量严格为一。
- 原图身份与文字已剥离，风格信息仍足以形成同一视觉宇宙。
- 输出只有可粘贴的 Prompt，没有分析过程、负向词或平台参数。
