# 现有技术冲突与可专利性风险分析框架

## 一、先做特征级对比

不要用整体“相似度”或重合百分比替代技术特征分析。先把每个候选保护点拆成必要技术特征，再逐篇核对证据。

| 对比项 | 记录内容 |
|---|---|
| 文献与日期 | 文献号、标题、公开日/优先权日、来源、法律状态核验情况 |
| 技术问题 | 是否解决相同或相近的具体技术问题 |
| 必要技术特征 | 对每个特征标记“明确公开 / 可能公开 / 未见公开 / 无法判断”，并引用段落、权利要求或附图位置 |
| 特征组合关系 | 单篇文献是否公开了同一组合，还是需要跨文献拼接 |
| 技术效果 | 区别特征是否产生有证据支持的技术效果 |
| 关键区别与未知项 | 尚未被文献覆盖的特征、事实缺口和需代理人判断的问题 |

### 单篇对比表

```text
| 必要技术特征 | 本方案 | 对比文件及证据位置 | 覆盖判断 | 备注/未知项 |
|---|---|---|---|---|
| 特征 A | [填写] | [段落、权利要求或附图] | 明确/可能/未见/不明 | [填写] |
```

搜索摘要或二手介绍只能支持候选筛选，不能作为“明确公开”的唯一依据；拿不到全文时应降级置信度。

## 二、风险等级

### 高风险

- 一篇日期合格的对比文件看起来明确覆盖某保护点的全部必要技术特征及其组合关系，可能直接影响新颖性；或
- 多篇高相关文件提供了清晰的组合线索，而区别特征及其技术效果缺少支撑，创造性风险明显。

### 中风险

- 对比文件覆盖多个核心特征，但仍存在可识别的区别特征；区别是否具有非显而易见性、是否产生技术效果尚不充分；或
- 关键日期、原文证据、分类号范围或方案材料存在影响结论的缺口。

### 低风险

- 已核验文献主要停留在相近领域、背景或局部特征，未见覆盖核心组合；并且
- 区别特征与技术效果有较清楚的材料支持。

“低风险”只表示在已完成检索范围内未发现强冲突，不等于可授权或不存在未检出的现有技术。

## 三、检索覆盖度与结论置信度

风险等级必须同时报告两个独立维度，不能混为一个分数：

### 检索覆盖度

- **高**：主要保护点均有中英文、同义词和分类号路径，覆盖相关主要来源，并追踪高相关文献的同族/引用。
- **中**：主要保护点已检索，但部分数据库、分类号路径、语言或非专利资料未覆盖。
- **低**：只做了少量关键词试探、仅看搜索摘要，或关键数据库不可访问。

### 结论置信度

- **高**：方案信息完整，关键文献原文和日期已核验，特征映射证据明确。
- **中**：核心判断有依据，但仍有材料缺口、解释空间或未核验信息。
- **低**：依赖不完整方案、摘要/二手来源、未知日期或大量推断。

## 四、未知项清单

每份报告单列未知项，不把未知默认为有利或不利：

1. 优先权日/拟申请日是否确认。
2. 必要技术特征、参数边界或替代实施方式是否完整。
3. 关键技术效果是否有实验或因果依据。
4. 高相关文献的全文、日期、同族和法律状态是否核验。
5. 哪些数据库、语言、分类号或非专利文献尚未覆盖。

## 五、建议类型

根据证据提出条件化建议，而不是固定“过审率”：

- 补充能证明区别特征及技术效果的实施细节或对比数据。
- 把候选保护点分为核心组合、可选限定和替代方案，供代理人设计层次。
- 对高风险特征重新检索、缩小或调整表述；是否修改正式权利要求由代理人决定。
- 对日期、权属、宽限期、创造性组合或法律状态等法律问题明确移交专业人员确认。
